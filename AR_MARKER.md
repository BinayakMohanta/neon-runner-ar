# Neon Runner AR - Marker Setup

## How to Use the Marker

1. **Download the HIRO marker** from AR.js:
   - Visit: https://cdn.jsdelivr.net/npm/ar.js@3/data/data/patt.hiro
   - Or use any standard AR.js pattern marker

2. **Print the marker** on a piece of paper (or display on a screen):
   - Size: At least 10cm x 10cm (4" x 4") recommended
   - Quality: Print clearly in color or B&W
   - Paper: Matte finish works best

3. **Point your phone camera** at the printed marker
4. **Tap "START SYSTEM"** when the game loads
5. **See the game world** appear on top of the marker!

## Playing the Game

- **Swipe left/right** on your screen to move your car
- **Or use the arrow buttons** at the bottom of the screen
- **Avoid red obstacles** to stay alive
- **Collect gold coins** to earn 100 points
- **Last as long as you can** while dodging and collecting!

You can use any AR.js compatible pattern marker:
- https://github.com/jeromeetienne/AR.js/tree/master/data/data

## Troubleshooting

- **Camera permission**: Grant camera permission when prompted
- **Lighting**: Ensure good lighting for marker detection
- **Marker visibility**: Keep the marker in camera view
- **Device support**: Works best on Android with Chrome/Edge
- **HTTPS required**: Game must be accessed over HTTPS for camera access

## Generate Your Own Marker

1. Visit: https://jeromeetienne.github.io/AR.js/three.js/examples/marker-training/examples/generator.html
2. Upload a custom image
3. Download the generated `.patt` file
4. Host it and update the game code to use your marker

Enjoy playing Neon Runner in AR!
